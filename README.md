# 防災備品チェックリスト

世帯人数・家族構成・備蓄日数を入力すると、必要な備品数を自動計算してくれる防災備蓄チェックリストアプリです。

---

## 概要

- 世帯人数を入力すると、飲料水・非常食などの必要数をリアルタイムで計算
- 子ども・女性・高齢者がいる世帯向けの追加備品を条件に応じて表示
- 3日分 / 7日分の切り替えに対応
- チェック済み備品を除外した「買い物リスト」を生成し、優先度（高・中・低）を設定可能
- 入力内容・チェック状態はブラウザの localStorage に自動保存
- 買い物リストの印刷に対応

---

## 使い方

### 1. 世帯情報を設定する

- **世帯人数** に人数を入力（1〜20人）
- **世帯の状況** に該当する条件（子どもがいる / 女性がいる / 高齢者がいる）をチェック
- **3日 / 7日** タブで備蓄期間を切り替える

### 2. 所持状況をチェックする

- 各備品のスイッチを「ある」に切り替えると、チェック済みとして記録される
- チェック状態はページを閉じても保存される

### 3. 買い物リストを確認する

- 「買い物リストを表示」ボタンをクリックすると、未チェック備品の一覧が表示される
- 各アイテムに優先度（高・中・低）を設定できる
- 「印刷する」ボタンでリストを印刷できる

### 4. リセット

- 「リセット」ボタンで全チェック・設定を初期化できる

## ローカルで実行する場合

ビルド不要の静的サイトです。`index.html` をブラウザで開くだけで動作します。

```bash
# リポジトリをクローン
git clone https://github.com/test-cloud-2026/bousai.git
cd bousai

# index.html をブラウザで開く（macOS の場合）
open index.html
```
